# README

React on Rails application with that exposes API endpoints for CRUD.

## Getting Started
#### Starting app locally
1. git clone https://github.com/Alan-w92/RoR-Coding-Challenge.git
2. rails db:create
3. rails db:migrate
4. rails db:seed
5. rails s
6. In a separate terminal run `yarn start`

#### Running Tests
1.rake
